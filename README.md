# Hamburger Chat

A chat interface featuring a hamburger menu navigation system.

## Tech Stack

- Next.js 14
- React 18
- TypeScript
- Tailwind CSS
- shadcn/ui
- Framer Motion

## Features

- 🍔 Intuitive hamburger navigation
- 💬 Real-time chat interface
- 📱 Responsive design
- 🌓 Dark mode support
- 🎨 Smooth animations

## Getting Started

```bash
# Install dependencies
pnpm install

# Run development server  
pnpm dev
```
